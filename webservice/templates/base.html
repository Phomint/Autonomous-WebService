Sobre

WebService

Este repositório tem como propósito servir como base de dados para a aplicação principal mobile, desenvolvido na linguagem Python, utiliza-se das frameworks Django e Django Rest Framework.

Como obter acesso ?

Para obter acesso aos arquivos json, é necessário saber qual tabela do banco quer ter acesso, insira o nome no final da url. phomint.pythonanywhere.com/'nomedatabela'

Exemplo:
phomint.pythonanywhere.com/autonomos

Para login foi usado Django Rest Auth então utilize os padrões 'user', 'login', 'logout', para registrar foi colocado uma extenção do Rest Auth mas continua como 'registrar'. Agora basta ler quais metodos estão disponiveis para essa tabela e utiliza-los.

Contribua para o projeto

Para contribuir acesse o código fonte do projeto, em seguida de um fork do projeto para sua máquina.

Agora que verificou os requisitos, instale uma Virtual Environment e inicialize ela dentro do projeto.

Windows
C:\Users\Name\Autonomous> python -m venv myvenv
Linux e OS X
$ python3 -m venv myvenv

Agora com o projeto em mãos, verifique se tem instalado em sua máquina os requisitos básicos do projeto.

  • certifi==2018.4.16
  • chardet==3.0.4
  • defusedxml==0.5.0
  • Django==2.0.7
  • django-allauth==0.36.0
  • django-rest-auth==0.9.3
  • djangorestframework==3.8.2
  • idna==2.7
  • oauthlib==2.1.0
  • Pillow==5.2.0
  • pkg-resources==0.0.0
  • python3-openid==3.1.0
  • pytz==2018.5
  • requests==2.19.1
  • requests-oauthlib==1.0.0
  • six==1.11.0
  • urllib3==1.23

Para instalar todos o requisitos de forma rápida, verifique se está no diretório do projeto e utilize o comando:

$ pip install -r requirements.txt

Depois de fazer suas alterações para o projeto volte a página do código fonte solicite um 'pull request'
