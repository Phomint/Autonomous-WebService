Sobre

WebService

Este repositório tem como propósito servir como base de dados para a aplicação principal mobile, desenvolvido na linguagem Python, utiliza-se das frameworks Django e Django Rest Framework.

Como obter acesso ?

Para obter acesso aos arquivos json, é necessário saber qual tabela do banco quer ter acesso, insira o nome no final da url. phomint.pythonanywhere.com/'nomedatabela'

Exemplo:
phomint.pythonanywhere.com/autonomos

Agora basta ler quais metodos estão disponiveis para essa tabela e utiliza-los

Contribua para o projeto

Para contribuir acesse o código fonte do projeto, em seguida de um fork do projeto para sua maquina.

Agora com o projeto em mãos verifique se tem instalado em sua maquina os requisitos básicos do projeto.

  • Django 2.0.7
  • Django Rest Framework 3.8.2
  • Pillow 5.2.0
  • pkg-resources 0.0.0
  • pytz 2018.5

Agora que verificou os requisitos, instale uma Virtual Environment e inicialize ela dentro do projeto.

Windows
C:\Users\Name\Autonomous> python -m venv myvenv
Linux e OS X
$ python3 -m venv myvenv

Depois de fazer suas alterações para o projeto volte a página do código fonte solicite um 'pull request'
