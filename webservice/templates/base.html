Sobre

WebService

Este repositório tem como propósito servir como base de dados para a aplicação principal mobile, desenvolvido na linguagem Python, utiliza-se das frameworks Django e Django Rest Framework.

Como obter acesso ?

Para obter acesso aos arquivos json, você pode navegar pela rotas definidas e obter se utilizar dos metodos GET, POST, UPDATE, DELETE. Em sua aplicação você definira quais caminhos e como vai obter e instanciar os dados.

ROTAS
  • api/autonomos/
  • api/autonomos/<#id>/

Para login foi usado Django Rest Auth então utilize os padrões 'user', 'login', 'logout', para registrar foi colocado uma extenção do Rest Auth mas continua como 'registrar'. Agora basta ler quais metodos estão disponiveis para essa tabela e utiliza-los.

Contribua para o projeto

Para contribuir acesse o código fonte do projeto, em seguida de um fork do projeto para sua máquina.

Agora que verificou os requisitos, instale uma Virtual Environment e inicialize ela dentro do projeto.

Windows, Linux e OS X

$ git clone https://github.com/Phomint/Autonomous-WebService
$ cd Autonomous-WebService
$ virtualenv myvenv -p python3
$ source myvenv/bin/activate
$ pip install -r requirements.txt
$ cd webservice
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver

Depois de fazer suas alterações para o projeto volte a página do código fonte solicite um 'pull request'
